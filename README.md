##  Características

### Diseño
- Tema oscuro premium (#0b0b0b)
- Paleta de colores con gradientes
- Responsive en todos los dispositivos
- Optimización de rendimiento

### Funcionalidad
- Navegación suave (smooth scrolling)
- Animaciones scroll (fade-in, parallax)
- Carrusel automático de certificados
- Efectos interactivos en hover
- Accesibilidad WCAG

### Secciones
1. **Hero** - Introducción con animaciones
2. **Sobre mí** - Información profesional
3. **Experiencia** - Timeline interactivo
4. **Certificaciones** - Carrusel automático
5. **Proyectos** - Grid de proyectos destacados
6. **Habilidades** - Categorías de competencias
7. **Contacto** - Links de contacto directo
8. **Footer** - Información adicional

##  Tecnologías

- HTML5
- CSS3 (Animations, Grid, Flexbox)
- JavaScript Vanilla (ES6+)
- Fuentes: Inter, Poppins (Google Fonts)

##  Inicio Rápido

1. Abrir \index.html\ en navegador
2. Navegación mediante menú o scroll
3. Hacer clic en proyectos para ver detalles
4. Usar links de contacto en sección de contacto

##  Personalización

### Cambiar Colores
Edita las variables CSS en \css/main.css\:

\\\css
:root {
--color-accent-primary: #667eea;
--color-accent-secondary: #764ba2;
--color-bg-primary: #0b0b0b;
--color-text-primary: #e8e8e8;
}
\\\
